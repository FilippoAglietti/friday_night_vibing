## Materials Checklist

### Before the Event
- [ ] Agenda finalized and sent to participants
- [ ] Slides / presentations created
- [ ] Worksheets and templates printed / shared
- [ ] Room / virtual setup confirmed
- [ ] Recording equipment ready (if applicable)
- [ ] Participant list confirmed
- [ ] Pre-work assignments sent

### Day-of
- [ ] Name tags / Zoom links ready
- [ ] Projector / screen sharing tested
- [ ] Backup materials available
- [ ] Snacks / coffee arranged (in-person)
- [ ] Feedback form ready
- [ ] Certificates prepared

### After the Event
- [ ] Feedback collected and analyzed
- [ ] Recordings / materials shared with participants
- [ ] Follow-up email sent (resources + next steps)
- [ ] Internal debrief completed
- [ ] Template updated based on feedback
