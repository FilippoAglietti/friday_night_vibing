# Syllabi Notion Template Pack

These templates are designed to be published on the [Notion Template Gallery](https://www.notion.so/templates).
Each one includes a "Made with Syllabi" footer link that drives traffic back to syllabi.online.

## Templates

### 1. Online Course Planner (`course-planner.md`)
**Target audience:** Course creators, coaches, consultants
**Use case:** Plan, structure, and track the creation of an online course
**Notion Gallery category:** Education, Productivity

### 2. Employee Onboarding Program (`employee-onboarding.md`)
**Target audience:** HR managers, team leads, startup founders
**Use case:** Structure a new hire onboarding program with modules and checklists
**Notion Gallery category:** HR, Team Management

### 3. Workshop & Bootcamp Template (`workshop-bootcamp.md`)
**Target audience:** Workshop facilitators, bootcamp organizers, community leaders
**Use case:** Plan intensive learning experiences with sessions, exercises, and materials
**Notion Gallery category:** Education, Events

### 4. Lead Magnet Mini-Course (`lead-magnet-course.md`)
**Target audience:** Marketers, solopreneurs, content creators
**Use case:** Create a free mini-course to capture emails and grow an audience
**Notion Gallery category:** Marketing, Content Creation

## Publishing Instructions

1. Create a new Notion page for each template
2. Copy the markdown content into Notion (it auto-formats)
3. Add a cover image using the Syllabi brand colors (violet gradient)
4. Submit to the Notion Template Gallery with the categories listed above
5. Include "Made with Syllabi — syllabi.online" in the description
